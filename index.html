Frequently Asked Questions (FAQs)

Have some questions on your mind? Here we added a few answers to frequently asked questions. If you want to know something is not listed here, just contact us.

PHPNuxBill is a robust and feature-rich freeRadius billing and management platform designed specifically for broadband internet service providers. It helps automate and streamline various aspects of managing an internet service business, including customer management, service provisioning, billing, and reporting.

Some of the key features of PHPNuxBill include customer self-signup, automated service provisioning, comprehensive billing and invoicing, bandwidth management, payment processing, reporting and analytics, and various integration capabilities.

PHPNuxBill is primarily targeted towards broadband internet service providers, such as ISPs, WISP (Wireless Internet Service Providers), and cable/fiber-optic companies, but it can also be useful for managed service providers (MSPs) or any business offering internet-based services.

Yes, PHPNuxBill is an open-source software, which means the source code is publicly available and can be accessed, modified, and distributed by the community.

The minimum system requirements for installing PHPNuxBill include a web server (Apache or Nginx), PHP 7.2 or higher, and a MySQL or MariaDB database. It can be deployed on-premises or in a cloud-based environment.

The installation process typically involves downloading the latest version of PHPNuxBill, setting up the database, configuring the web server, and running the installation wizard. Detailed installation guides are available in the PHPNuxBill documentation.

PHPNuxBill supports the management of various internet-based services, including broadband internet (wired and wireless), VoIP, IPTV, Hotspot, Static IP and even bundled services that combine multiple offerings.
